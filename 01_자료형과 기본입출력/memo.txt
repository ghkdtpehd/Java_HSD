[주석]
// 				: 뒤부터 쭉 주석
/* */ 			: 범위내 주석

[단축키]
ctrl / 			: 한줄 주석
ctrl shift c 	: 줄 주석 (범위지정후 주석 가능)
ctrl c 			: 복사
ctrl v 			: 붙여넣기
ctrl alt 아래 	: 줄 복사 (블록 복사)
alt 방향키 		: 줄 이동 (블록 복사)
ctrl d 			: 줄 지우기 (블록 제거)

[문자 string]
'' 				: 문자 상수 - 1개의 문자만 들어감
"" 				: 문자열 상수

문자열끼리의 '+'는 연결의 의미이다.
문자열 + 상수일시 결과는 문자열이된다.
문자열 + 상수 + 상수 일 시  상수를 묶어주면 (ex (상수 + 상수))  먼저 계산후 문자열로 바뀐다.

변수명은 영문자 또는 _로 시작해야한다. 

float는 변수로 지정할시 뒤에 f를 붙여야한다. ex) 1.23f 
또한 소숫점 7째 자리수까지 나오며, 반올림한다.

크기 double > float > int

[서식문자]
%d 정수
%c 문자
%f 실수
%.n 소숫점 자리수를 n까지만 표시한다.
%5d 5개의 자리 표시
\n 줄바꿈
\t tab키
